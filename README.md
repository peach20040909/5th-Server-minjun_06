# 5th-Server 6주차 - 게시판 CRUD API

스프링 부트 + JPA로 만든 게시판 REST API입니다.
6주차 스터디 과제로 진행했습니다.

## 기능
- 게시글 작성 / 조회 / 수정 / 삭제 (CRUD)
- REST API 방식 (GET/POST/PUT/DELETE)
- Entity와 DTO 분리

## 기술 스택
- Java 17, Spring Boot
- Spring Data JPA, H2 Database

## 실행 방법
1. BoardApplication 실행
2. http://localhost:8080/api/posts 에서 API 확인
